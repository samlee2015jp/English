パソコン上での再生や，携帯音楽プレーヤーなどでお使いいただけます。

◇ MP3ファイルの利用方法 ◇

収録されているMP3ファイルは，以下のようにご利用いただけます。
・パソコンで直接聴く
・Windows Media Player や iTunes などの音楽ソフトを経由し，携帯音楽プレーヤーに取り込んで聴く

ここでは，Windows Media Player と iTunes での再生方法をご紹介します。

◆Windows Media Player での再生
　(1) MP3ファイルをCD-ROMからパソコン上にコピーしてください。
　(2) Windows Media Playerを起動してください。
　(3) Windows Media Playerの左上にある「ファイル」メニューから［開く］を選択してください
　(4) コピーしたファイルのすべてあるいは再生したいものだけを選択し，「開く」をクリックしてください。


◆iTunesでの再生
　(1) MP3ファイルをCD-ROMからパソコン上にコピーしてください。
　(2) iTunesを起動してください。
　(3) iTunesの左上にある「ファイル」メニューから以下を選択してください。
　　・Windowsの場合：［フォルダーをライブラリに追加］
　　・Macの場合：［ライブラリに追加］
　(4) コピーしたファイルのすべてあるいは再生したいものだけを選択し，「OK」をクリックしてください。


※上記以外のアプリケーションソフトのご利用方法や携帯音楽プレーヤーへの転送方法については，そのソフトのマニュアルをご確認ください。
